# Python Log Review Tool

## Overview
This project is a simple Python-based log review tool created to practice basic automation and security monitoring concepts. It reads a sample log file and identifies lines containing potentially suspicious activity such as failed logins, denied access attempts, errors, or unauthorized actions.

## Purpose
The goal of this project is to build foundational Python and automation skills that support cybersecurity and DevSecOps workflows.

## Technologies Used
- Python
- Text-based log file analysis

## Key Features
- Reads a log file
- Searches for suspicious keywords
- Prints flagged log entries for review

## Example Use Cases
- Failed login monitoring
- Basic error review
- Unauthorized access detection
- Introductory automation practice

## Skills Demonstrated
- Python fundamentals
- Security monitoring concepts
- Basic automation
- Log analysis workflow

## Future Improvements
- Export results to CSV
- Add timestamps and severity levels
- Expand keyword matching
